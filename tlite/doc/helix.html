Helix Utility Functions



These routines calculate the four momentum of a particle from its helix parameters or the vertex constrained parameters(theta, phi, and curvature). For the helix parameters the momentum is calculated at the point of closest approach to the reference point, so one might want to call vcreref before.

Functions:
  • tliphel(par,cov,bfieldz,mass,mode,p,pcov)

    The momentum is calculated at the point of closest approach to the reference point, so one might want to call vcreref before.

    input variables:
    • par
      real(5); an array holding the parameters of the track.
      The track parameters follow the ZEUS convention(ΦH, Q/R, QDH, ZH,cot θ).
    • cov
      real(15); an array holding the covariance matrix elements of the track parameters.
    • bfieldz
      real; the z component of the magnet field in kGauss
    • mass
      real; mass of the particle
    • mode
      integer; compute covariance matrix if mode > 1
    output variables:
    • p
      double precision(4); the four momentum at the point of closest approach to the reference point of the helix.
      Order: Px,Py,Pz,E
    • pcov
      double precision(10); the covariance matrix of the calculated four momentum
      format:
      1247
      358
      69
      10
  • tlipnewpar(par,bfieldz,mass,p)

    input variables:
    • par
      real(5); an array holding the vertex constrained parameters of the track.
      These parameters are &theta,&Phi, and Q/R.
    • bfieldz
      real; the z component of the magnet field in kGauss
    • mass
      real; mass of the particle
    output variables:
    • p
      double precision(4); the four momentum
      Order: Px,Py,Pz,E
