Kinematic fits



Functions:
  • tlifitm(parrm,covrm,m,chi2,err)

    Perform mass constraint fit on a momentum 4-vector.

    input variables:
    • parrm
      double precision(4); an array holding the four momentum (Px,Py,Pz,E)
    • covrm
      double precision(10); an array holding the covariance matrix elements of the four momentum.
      format:
      1247
      358
      69
      10
    • m
      real; tabulated mass of the particle.
    output variables:
    • parrm
      double precision(4); constrained (px,py,pz,E) of resonance
    • covrm
      double precision(10); constrained packed covariance matrix format:
      1247
      358
      69
      10
    • chi2
      real; χ² of the mass constraint fit.
    • err
      integer; error flag (0 if OK).
