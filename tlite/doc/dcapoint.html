Distance of Closest Approach from a Point



These routine returns the distance of closest approach of a track and a point in two dimensions (x,y).
Functions:
real d
  • d = tlidcapoint(parIn,covIn,xpoint,ypoint,covmode,eDca)
    input variables:
    • parIn
      real(5); an array holding the parameters of the track.
      The track parameters follow the ZEUS convention(ΦH, Q/R, QDH, ZH,cot θ).
    • covIn
      real(15); an array holding the covariance matrix elements of the track parameters.
    • xpoint
      real; x-coordinate of the point
    • ypoint
      real; y-coordinate of the point
    • covmode
      integer; compute error of DCA if covmode > 1
    output variables:
    • eDCA
      real; error on the distance of closest approach
