VxLite - A Lightweight Vertexing Package



Functions:
  • vxlitebs(beamwidth,vertex,vertexcov,ierr)
    vxlitebs uses bstilt(the beam spot GAF) to construct an estimate of the primary vertex. The beam tilt is included in the covariance matrix.

    input variables:
    • beamwidth
      real(2); beam width in x and y to use (cm)
      if both values are 0 the GAF values are used. It is recommened to overwrite these with e.g. 0.0120 0.0030 by setting beamwidth.
    output variables:
    • vertex
      real(3); Cartesian coordinates in cm of the constructed vertex.
    • vertexcov
      real(6); elements of the covariance matrix of the constructed vertex.
      The order of the elements is:
      124
      35
      6
    • ierr
      integer; error code from bstilt
      0 = OK
      1 = GAF unavailable
      2 = run not in GAF
