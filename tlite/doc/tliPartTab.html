The tliPartTab Class

Purpose

The purpose of this class is to provide an interface to the ZEUS particle table that is easy to use from C++.

Implementation

The class is implemented as a singleton. Upon its first invocation, an instance is created which loads its information from the FMCPrt and FMCPcd tables.

Member functions

  static tliPartTab* Instance();   // give pointer to singleton instance
  int getFmcId(const string& partName);   // convert particle name to integer particle id
  int getAntiFmcId(const string& partName);  // find id of anti-particle
  string getAntiName(const string& partName);   // give name of anti-particle for given name of particle
  static int getFmcDaug(const int fmckinId,vector<int>& output);  // expand mother-daughter relations in FMCKin table
