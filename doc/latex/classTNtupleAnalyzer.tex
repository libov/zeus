\subsection{Detailed Description}
Base class for \doxyref{TMini\-Ntuple\-Creator}{p.}{classTMiniNtupleCreator} and \doxyref{TMini\-Ntuple\-Analyzer}{p.}{classTMiniNtupleAnalyzer}. 

Implements functionality which is common for both cases: analysis (both on d\-Cache and minintuples) and creation of minintuples.
