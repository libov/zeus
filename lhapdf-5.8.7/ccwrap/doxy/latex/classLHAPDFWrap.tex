\subsection{Detailed Description}
Wrapper class used to contain all the wrapper functions. \begin{Desc}
\item[\bf{Deprecated}]The class-based C++ wrapper on \doxyref{LHAPDF}{p.}{namespaceLHAPDF} will be retired in a forthcoming release of \doxyref{LHAPDF}{p.}{namespaceLHAPDF} in favour of the namespaced wrapper declared in {\tt \doxyref{LHAPDF.h}{p.}{LHAPDF_8h-source}}. Please convert client code which uses the class interface to use the new interface instead. Typically, this will just involve changing the header include from {\tt \char`\"{}LHAPDF/LHAPDFWrap.h\char`\"{}} to {\tt \char`\"{}LHAPDF/LHAPDFWrap.h\char`\"{}}, changing any constructors to initialisation functions, and replacing {\tt \doxyref{LHAPDFWrap}{p.}{classLHAPDFWrap}} objects with an {\tt \doxyref{LHAPDF}{p.}{namespaceLHAPDF}} namespace. For example, \end{Desc}


\begin{Code}\begin{verbatim}  #include "LHAPDF/LHAPDFWrap.h"
  LHAPDFWrap pdf = LHAPDFWrap("MRST2004qed.LHgrid", 0);
  pdf.getDescription();
\end{verbatim}\end{Code}

 would be replaced by 

\begin{Code}\begin{verbatim}  #include "LHAPDF/LHAPDF.h"
  LHAPDF::initPDFByName("MRST2004qed.LHgrid", 0);
  LHAPDF::getDescription();
\end{verbatim}\end{Code}
